Claude Bull — Product POC Blueprint

Features · Inputs & Outcomes · Tech Stack · Components · Style

Core Features (POC Scope)

MVP
Ticker Research Wizard
Step-by-step guided flow. User enters AAPL, Claude runs all 9 analysis steps sequentially. Each step shows AI output + source citations.
MVP
SEC EDGAR Ingestion
Auto-fetch 10-K, 10-Q, and earnings transcripts from SEC's public API. No auth needed. Parse and chunk for Claude context window.
MVP
Citation Engine
Every AI claim links to an exact filing paragraph. Inline [source] tags with hover preview. Zero hallucination tolerance enforced by prompt rules.
MVP
Research Memo Output
Final output is a structured markdown memo. Exportable as .md or PDF. Sections: Business, Financials, Moat, Valuation, Bear Case, Decision.
MVP
BYOK API Key Setup
First-run prompt asks for Anthropic API key. Stored encrypted in localStorage. Never sent to any server. Usage cost estimates shown per analysis.
MVP
Bear Case Gate
Step 7 is mandatory before any "buy" signal. AI plays devil's advocate using the same source filings. Contradictions vs bull thesis highlighted.
V2
Decision Journal
Log your final decision (buy/watch/pass) with reasoning. Timestamped. Revisit after earnings to compare thesis vs reality.
V2
Quarterly Update Diff
On new earnings, re-run affected steps and highlight what changed vs prior quarter. Red flags thesis-breaking changes automatically.
V2
Portfolio View
Dashboard of all researched tickers. Progress, completion status, last updated, quick-access to any memo. Watchlist management.

Design Principles

Terminal Meets SaaS
Dark, dense, data-forward. Bloomberg terminal energy with modern SaaS polish. Monospace for all financial data.
Citations First
Source tags are visually prominent (green). Unsourced claims don't exist. Trust is earned through transparency.
One Step at a Time
Wizard-style flow. No overwhelm. Each step gets full screen focus. Progress is always visible.
